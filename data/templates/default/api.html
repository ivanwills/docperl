Note: This API is just a guess on the meaning of the subroutines in the file.
Class Methods are assumed be in one of the following forms: 
sub class_method {
	my $class = shift;
	...
}

or
sub class_method {
	my $caller = shift;
	...
}
Object Methods are assumed to be in one of the following forms: 
sub object_method {
	my $self = shift;
	...
}

or
sub object_method {
	my $this = shift;
	...
}

or
sub object_method {
	my ( $self, ... ) = @_;
	...
}

or
sub object_method {
	my ( $this, ... ) = @_;
	...
}
